# vue-memo-app

This is a simple memo application powered by Vue.js

You can do the following

Add memo
View and Edit memo
Delete memo

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
